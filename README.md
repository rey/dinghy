# dinghy

## About

This is a dummy repo that aims to provide a generic environment to run Middleman
in an Ubuntu VM running on Vagrant with VirtualBox.

## Prerequisites

* VirtualBox (4.3.12)
* Vagrant (1.5.1)

## Get started

1. Clone the repo

        git clone https://github.com/rey/submarine.git ~/your-project

2. `cd ~/your-project`
3. `vagrant up`
4. `vagrant ssh`
